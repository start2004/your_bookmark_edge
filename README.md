# 你的书签

### 产品说明

- 简洁。简洁平铺书签，方便快速找到链接。
- 流畅。轻量级应用，1秒加载完成。
- 安全。无任何网络请求，保护隐私。

- Concise. Neatly arrange bookmarks for easy and quick link access.
- Smooth. Lightweight application, loads in 1 second.
- Secure. No network requests, privacy protected.

### 单一用途说明

- 展示用户的收藏夹，平铺书签链接，方便用户快速找到自己想要访问的链接
- 提供多种搜索引擎，便于用户选择
